# Why Most Lottery Winners Go Broke — Image Prompt Pack (105 frames, MS Paint stickman style)

## How to use
1. Paste the STYLE BLOCK below at the start of every generation request (or once per chat session, then say "same style" before each scene).
2. Generate frames in batches of 5–10 so the style stays consistent.
3. After the first good image, attach it as a reference ("match this exact style") for all following frames.
4. All images: horizontal 16:9, white background.

## STYLE BLOCK (prepend to every prompt)
Extremely simple amateur MS Paint style drawing, as if drawn quickly by hand by someone who cannot draw. White background, thick uneven shaky black outlines, stick figures with round heads, dot eyes and line bodies. Flat colors only (red, yellow, green, blue, gray, brown, orange, used sparingly), mostly empty white space. No shading, no gradients, no 3D, no realistic or cinematic lighting, no professional vector art, no anime, no Disney style, no polished illustration. Crude, funny, intentionally clumsy doodle. Horizontal 16:9 YouTube frame, clean centered composition, nothing important cropped. Any text inside the image must be short, hand-written looking, spelled correctly and easy to read.
